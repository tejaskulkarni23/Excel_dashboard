The Zomato Sales Analysis Dashboard is an interactive Microsoft Excel project designed to analyze and visualize restaurant sales data. The main objective of this project is to transform raw data into meaningful business insights that help understand customer behavior, sales performance, and overall business growth.

This dashboard is built using various Excel features such as Pivot Tables, Pivot Charts, Slicers, Conditional Formatting, Data Cleaning, and Data Visualization techniques. The dataset is organized and processed to create a user-friendly interface where users can quickly explore key performance indicators (KPIs) and make data-driven decisions.

The dashboard provides insights into important metrics such as:

📊 Total Sales
🍔 Total Orders
👥 Customer Count
💰 Average Order Value
🌍 City-wise and Region-wise Performance
🏪 Restaurant Category Analysis
📅 Monthly and Yearly Sales Trends
⭐ Top Performing Restaurants and Food Categories

Interactive slicers allow users to filter the data based on different dimensions, making the dashboard dynamic and easy to navigate. The visualizations help identify sales patterns, peak order periods, and high-performing locations, enabling better strategic planning.

This project demonstrates practical knowledge of Excel for Data Analysis and Business Intelligence. It showcases the ability to clean and prepare datasets, build interactive dashboards, and present complex information in a simple and visually appealing format.
